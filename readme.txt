カウンター

製作者：Matsu-Kiham
バージョン：1.00

Webブラウザ上で動くカウンターアプリです．
カウントした期間と回数から，
出来事の起こる確率と平均発生回数を計算します．

導入方法：
php7.1x以上が利用可能なサーバーにcounter.phpファイルを
置いてください．このファイルと同一の階層にdataフォルダ
を置いてください．


操作方法：
新規作成 -> 例えば，N:晴の日，という風に”N:”の文字列に
続いてカウントしたい項目の名前を画面下部の送信フォーム
に入力して送信ボタンを押します．すると，画面上部にある
背景がグレーのフォーム上にx:晴れの日という項目名（xは
項目番号）とカウントボタン（ピンクのボタン）が追加され
ます．ピンクのボタンを押すと，押した日付と時刻が記録されます．
なお，グレーのフォーム上には初めから
1行目は触らないでね(^^)，という項目が入っていますが，
この項目は使用しないで，2行目以降を使ってください．

編集 -> 例えば，2:晴の日，3:雨の日という項目がグレーの
フォーム上にあるとします．このとき，3:雨の日を2:晴の日
の上に持っていく，すなわち，2:雨の日，3:晴の日とする場合，
E:3 2と送信フォームに入力して送信ボタンを押します．
上に持っていきたい数字，半角スペース，下に置きたい数字，
という順番で入力します．

削除 -> 例えば，2:晴の日という項目を削除したい場合，D:2
と送信フォームに入力して送信ボタンを押します．

詳細 -> 例えば，2:晴の日という項目がグレーのフォーム上に
あるとします．このとき，O:2と送信フォームに入力して送信
ボタンを押すと，送信フォームの下部に以下の内容が表示され
ます．

項目名
カウントボタンが押された合計回数
カウントボタンが最初に押されたときから最後に押されたとき
までの期間（秒，時間，日，月，年の順に表示）
項目が1秒あたりに起こる確率
1，3，6，12時間以内に少なくとも1回項目が起こる確率
1，3，7，14日以内に少なくとも1回項目が起こる確率
1，3，6月以内に少なくとも1回項目が起こる確率
1，3，10，30，50年以内に少なくとも1回項目が起こる確率
1，3，6，12時間以内に項目が起こる平均回数
1，3，7，14日以内に項目が起こる平均回数
1，3，6月以内に項目が起こる平均回数
1，3，10，30，50年以内に項目が起こる平均回数
ログ

※(項目が1秒あたりに起こる確率)=(カウントボタンが押された合計回数)/(カウントボタンが最初に押されたときから最後に押されたときまでの期間[秒])
※(t時間以内に少なくとも1回項目が起こる確率)=1 - (1 - (項目が1秒あたりに起こる確率))^(t*3600)
※(t日以内に少なくとも1回項目が起こる確率)=1 - (1 - (項目が1秒あたりに起こる確率))^(t*86400)
※(t月以内に少なくとも1回項目が起こる確率)=1 - (1 - (項目が1秒あたりに起こる確率))^(t*2592000)
※(t年以内に少なくとも1回項目が起こる確率)=1 - (1 - (項目が1秒あたりに起こる確率))^(t*31536000)
※(t時間以内に項目が起こる平均回数)=t*3600
※(t日以内に項目が起こる平均回数)=t*86400
※(t月以内に項目が起こる平均回数)=t*2592000
※(t年以内に項目が起こる平均回数)=t*31536000
